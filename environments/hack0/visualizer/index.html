Rubik's Cube Visualizer

API Access

Get the current cube state via API: GET http://localhost:3000/api/cube-state

Note: The cube state is automatically updated on the server whenever the cube changes!

Control the Cube

Send moves to the cube: POST http://localhost:3000/api/move with JSON body: {"move": "R2"}

Supported moves: U, D, L, R, F, B (clockwise), U', D', L', R', F', B' (counterclockwise), U2, D2, L2, R2, F2, B2 (double)
